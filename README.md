# LB2-ZuzkovRoman
Вариант: 
## Использование
1. Сначала запустите server.py
> Сервер начнет слушать подключения и покажет сообщение "Сервер запущен на 127.0.0.1:5555"
2. Запустите клиент client.py

## Использование:
В открывшемся окне введите ваш ID (любое число или текст, например: "user1")

Введите ID собеседника (например: "user2")

Откроется окно чата с историей сообщений

Введите новое сообщение и нажмите Enter или кнопку "Отправить"

Сообщение автоматически сохранится и отобразится в чате

Кнопки Выйти и Сменить пользователя возвращают на предыдущие страницы.

> На любой странице можно использовать Enter для отправки введенного айди, или сообщения.
